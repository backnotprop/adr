# ADR

Explicit ADR (architecture decision record) workflow router skill.

## Install

```bash
npx skills add backnotprop/adr
```

List available skills first:

```bash
npx skills add backnotprop/adr --list
```

Local development install:

```bash
npx skills add . --list
npx skills add . -y
```

## Skill

- `adr`: explicit-only router for `adr research`, `adr synthesize`, `adr spec`, `adr adr`, `adr intent`, `adr implement`, `adr self-review`, and `adr recap`.
